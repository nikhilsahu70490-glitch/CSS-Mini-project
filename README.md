# 📸 Responsive Sidebar Menu Using HTML & CSS

A modern and responsive sidebar navigation menu built using **HTML5** and **CSS3**. The project uses the **CSS Checkbox Hack** to create an interactive sidebar without any JavaScript.

## 🚀 Features

- Responsive full-screen layout
- Sidebar navigation menu
- Pure HTML & CSS (No JavaScript)
- CSS Checkbox Hack for menu toggle
- Smooth hover animations
- Font Awesome icons
- Google Fonts integration
- Background image with full-screen cover
- Social media icons section
